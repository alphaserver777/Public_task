### 1. Создание топологии

* Построить топологию, как указано в задании. Используйте маршрутизатор и коммутатор, затем соедините их с двумя ПК, как показано на схеме.
* Назовите маршрутизатор и коммутатор вашей фамилией, используя команду `hostname`.

### 2. Назначение IP-адресов компьютерам

* На каждом ПК вручную настройте IP-адреса, как указано на схеме. Например:

* PC1: 192.168.10.2/24
* PC2: 192.168.20.2/24
* После этого сделайте скриншот, чтобы подтвердить правильность настройки.

### 3. Создание VLAN и назначение интерфейсов

* На коммутаторе создайте VLAN 10 и VLAN 20:

```text
Switch> enable
Switch# configure terminal
Switch(config)# vlan 10
Switch(config-vlan)# name VLAN10
Switch(config-vlan)# exit
Switch(config)# vlan 20
Switch(config-vlan)# name VLAN20
Switch(config-vlan)# exit
```
* Назначьте порты коммутатора на эти VLAN:

```text
Switch(config)# interface range fa0/1 - 2
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 10
Switch(config-if-range)# exit

Switch(config)# interface range fa0/3 - 4
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 20
Switch(config-if-range)# exit
```

### 4. Изменение имен маршрутизатора и коммутатора

* На **каждом** устройстве выполните команду `hostname`, чтобы изменить имя:

```text
Router> enable
Router# configure terminal
Router(config)# hostname <Фамилия>R1   ← свое имя
<Фамилия>R1(config)# end
```

Аналогично выполните на коммутаторе:

```text
Switch> enable
Switch# configure terminal
Switch(config)# hostname <Фамилия>Switch
<Фамилия>Switch(config)# end
```

### 5. Настройка маршрутизации "Router on a stick"

* На маршрутизаторе создайте интерфейсы под каждую VLAN:

```text
Router> enable
Router# configure terminal
Router(config)# interface gig0/0/0.10
Router(config-if)# encapsulation dot1Q 10
Router(config-if)# ip address 192.168.10.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit

Router(config)# interface gig0/0/0.20
Router(config-if)# encapsulation dot1Q 20
Router(config-if)# ip address 192.168.20.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
```

### 6. Проверка пинга

* На ПК, подключенных к VLAN 10 и VLAN 20, выполните пинг друг к другу, чтобы убедиться, что они могут общаться через маршрутизатор.
* Сделайте скриншот успешного пинга.

### 7. Сохранение конфигураций

* На маршрутизаторе и коммутаторе сохраните настройки в энергонезависимую память:

```text
Router# copy running-config startup-config
Switch# copy running-config startup-config
```

### 8. Отправка отчета

* Подготовьте отчет, включающий конфигурационные файлы и скриншоты, и отправьте его на проверку через журнал.
