# Вопрос 3: Мультиплексирование и модуляция

---

#### Подробный ответ:

**1. Мультиплексирование**

Мультиплексирование — это технология объединения нескольких сигналов для передачи по одному каналу связи, чтобы экономить ресурс канала. Основные виды:

1. **Частотное мультиплексирование (FDM — Frequency Division Multiplexing)**

* Каждый сигнал передаётся на своей частоте (поднесущей).
* Используется в аналоговой и радиосвязи, кабельном ТВ.
* Пример: несколько радиостанций в одном кабеле.

2. **Временное мультиплексирование (TDM — Time Division Multiplexing)**

* Канал делится на временные интервалы (слоты).
* Каждый сигнал передаётся по очереди в своем слоте.
* Пример: цифровые телефонные линии, SDH/SONET.

3. **Кодовое мультиплексирование (CDM/CDMA — Code Division Multiplexing)**

* Каждому сигналу присваивается уникальный код, позволяющий одновременно передавать их в одной полосе частот.
* Пример: мобильные сети 3G, GPS.

4. **Мультиплексирование по длине волны (WDM — Wavelength Division Multiplexing)**

* Для оптоволокна: каждый сигнал передаётся на своей длине волны света.
* Пример: DWDM в магистральных оптических линиях.

---

**2. Модуляция**

Модуляция — это процесс изменения характеристик несущего сигнала для передачи информации. Она позволяет передавать данные по среде, которая не поддерживает прямую цифровую передачу.

**Основные виды модуляции аналогового сигнала:**

1. **Амплитудная (AM — Amplitude Modulation)**

* Меняется амплитуда несущей в соответствии с сигналом.
* Пример: AM-радио.

2. **Фазовая (PM — Phase Modulation)**

* Меняется фаза несущей.
* Пример: некоторые системы передачи данных, цифровые модемы.

3. **Частотная (FM — Frequency Modulation)**

* Меняется частота несущей.
* Пример: FM-радио.

**Дополнительно:**

* Для цифровых сигналов применяются **цифровые виды модуляции**:

* ASK (Amplitude Shift Keying), FSK (Frequency Shift Keying), PSK (Phase Shift Keying), QAM (Quadrature Amplitude Modulation).
* Цель: более устойчивое и эффективное использование канала, меньшая чувствительность к шуму.

---

**Краткий ответ для Anki:**

* **Мультиплексирование:** FDM (частоты), TDM (время), CDM (коды), WDM (длины волн).
* **Модуляция:** изменение несущего сигнала для передачи информации.

* **AM** — амплитуда, **FM** — частота, **PM** — фаза.
* **Цифровая модуляция:** ASK, FSK, PSK, QAM.
